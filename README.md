# Regions (Python Port)

This is a (work in progress) Python port of [Regions](https://github.com/pietro-bozzo/Regions).

This repository provides a functional port of Regions, preserving core algorithms and structural design while adapting interfaces to Pythonic conventions where appropriate.

## Installation

This package is not yet distributed via PyPI or any package index. Installation is currently limited to local, editable installs, e.g., on Linux:
```
$ cd /path/to/RegionsPython/fmatoolbox
$ pip install -e . # install fmatoolbox in currently active python environment
$ cd /path/to/RegionsPython/regions
$ pip install -e . # same for regions
```
